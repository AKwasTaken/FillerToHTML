Filler Language Guide

What is Filler?

Filler is a simple scripting language created for The Coding Club of IISER Thiruvananthapuram to generate structured HTML content for events. It simplifies the process of creating formatted content with various elements like text, images, and links.

Basic Structure

title{Event Title}

topic{Main Topic}

body{
    Your content here
    bold{Important text}
    pointers{
        Point 1
        Point 2
    }
}

Filler Language was created by Aneeth Kumaar

Available Functions

title

Creates a title heading for your content

title{Alumni Discussion}
Creates a main heading: "Alumni Discussion"

topic

Defines the main topic or subtitle

topic{Unix and their Advantages}
Creates a subtitle: "Unix and their Advantages"

body

Contains the main content with various formatting options

body{ // Regular text Welcome to our workshop! // Large bold text bigbold{Join us for an exciting session} // Bold text bold{Important information:} // Italic text italic{Registration closes soon} // Bold and italic text bolditalic{Don't miss out!} // Large, bold, and italic text bigbolditalic{Special Announcement} // Bulleted list pointers{ Basic concepts Advanced topics Hands-on practice } // Image image{https://example.com/workshop.jpg} }
Each formatting option can be used independently or combined within the body.

info

Displays event details in a structured format

info{ // Event date Date: 17-12-24 // Event time Time: 2200 Hours // Location Venue: LHC Venus // Speaker information Speakers: Ashutosh Kumar and Krishna Kumar Singh }
Creates a formatted information box with event details.

rectangle

Creates a speaker/profile card with details

rectangle{ // Profile image URL image: https://example.com/image.jpg // Full name name: Ashutosh Kumar // Batch information Batch: 2019 // Department/Institution Dept: Faculty of Information Technology, University of Jyväskylä // Area of expertise Research Focus: Quantum Information and Computation // Brief biography About: Ashutosh is a trailblazer in the field of quantum computing }
Creates a profile card with image and detailed information.

divider

Adds a horizontal line to separate content sections

divider{}
Inserts a horizontal line for visual separation.

link

Creates a hyperlink with custom text

// Basic link with text and URL link{Join our WhatsApp Group url{https://chat.whatsapp.com/...}} // Can be used within body body{ Click here to register: link{Register Now url{https://forms.example.com}} }
Creates clickable links with custom text.

Complete Event Example

title{Tech Talk Series}

topic{Machine Learning Workshop}

body{
    bigbold{Join us for an exciting workshop!}
    
    We are thrilled to announce our upcoming workshop on Machine Learning fundamentals.
    
    bold{What you'll learn:}
    pointers{
        Basic ML concepts
        Python for ML
        Hands-on exercises
        Real-world applications
    }
}

info{
    Date: 25-03-24
    Time: 1400 Hours
    Venue: Online (Zoom)
    Speaker: Dr. Jane Smith
}

divider{}

rectangle{
    image: https://example.com/speaker.jpg
    name: Dr. Jane Smith
    Batch: 2015
    Dept: Computer Science, Stanford University
    Research Focus: Deep Learning
    About: Leading researcher in AI with 10+ years of experience
}

body{
    bolditalic{Don't miss this opportunity!}
    
    link{Register Now url{https://register.example.com}}
}

Best Practices

  • Add empty lines between sections for better readability
  • Indent nested content for clearer structure
  • Ensure all brackets {} are properly closed
  • Use appropriate formatting for different types of content
  • Keep the structure consistent across all your scripts
